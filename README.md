# hitchhiker
Telegram automation for hitching rides

[![Build Status](https://travis-ci.org/yehonatanz/hitchhiker.svg?branch=master)](https://travis-ci.org/yehonatanz/hitchhiker)

Listens to telegram messages with a whitelist of chats and streets, replies to the first matching message and exiting.
Built using Telethon, currently manually deployed as cron on an AWS machine.
